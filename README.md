# Credit Risk Prediction

## Project Overview
This project predicts whether a loan applicant is a credit risk using machine learning.

## Dataset
German Credit Dataset

## Models Used
- Logistic Regression
- Random Forest

## Results
| Model | Accuracy |
|------|------|
| Logistic Regression | 0.74 |
| Random Forest | 0.73 |

## Visualization
Model comparison and confusion matrices were used to evaluate performance.
